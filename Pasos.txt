-----IMPORTAR PROYECTO MAVEN EXISTENTE
IR AL MENU FILE->IMPORT->EXISTING MAVEN PROJECTS
UBICAR LA CARPETA DONDE ESTA EL PROYECTO, SELECCIONAR LA CARPETA E IMPORTAR TODAS LAS
CARPETAS.(EN WINDOWS NO IMPORTAR LA CARPETA SEMILLERO WEB Y PODER ELIMINAR LA CARPETADE SEMILLERO PADRE)

-----PASOS PARA CREAR BACKEND EN JAVA
1. CREAR LA TABLA EN ORACLE CON SUS RESPECTIVOS CAMPO, CONSTRAINT Y SECUENCIAS.
2. CREAR LA CLASE ENTIDAD EN EL PAQUETE RESPECTIVO DONDE SE ENCUENTRAN LAS ENTIDADES
3. CREAR LA CLASE ENTIDADDTO EN EL PAQUETE RESPECTIVO DONDE SE ENCUENTRAN LAS ENTIDADESDTO
4. CREAR LA INTERFACE EN EL PAQUETE RESPECTIVO DONDE SE ENCUENTRAN CON LAS OPERACIONES DEL CRUD
5. CREAR LA CLASE BEAN EN EL PAQUETE RESPECTIVO DONDE SE ENCUENTRAN CON LAS OPERACIONES DEL CRUD REFERENCIADAS EN LA INTERFACE
6. CREAR LA CLASE REST EN EL PAQUETE RESPECTIVO DONDE SE ENCUENTRAN DEFINIENDO EL PATH(RUTA DE ACCESO URL) Y LAS 
   OPERACIONS DEL CRUD REFERENCIADAS EN LA INTERFACE
